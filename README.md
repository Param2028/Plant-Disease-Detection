Plant Disease Detection
An AI-powered system that identifies diseases in plant leaves and recommends fertilizers. Users can upload leaf images, and the system predicts the disease using deep learning models (MobileNetV2 & InceptionV3) with high accuracy.

Features
Upload plant leaf images for disease detection
Automatic disease prediction with deep learning
Fertilizer recommendation for detected disease
User-friendly web interface

Technologies Used
Python, TensorFlow / Keras – Deep learning models
OpenCV, NumPy, Pandas – Image processing & data handling
HTML, CSS, JavaScript, Flask – Frontend & backend web interface
Matplotlib & Seaborn – Model performance visualization

Purpose
Helps farmers detect plant diseases early, reduce crop loss, and apply the right fertilizer to improve yield.
